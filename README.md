## Core Features

### Event & Ticket Management
- Create, edit, publish, and delete events
- Ticket category setup (VIP, General, Early Bird, etc.)
- Real-time ticket stock control across all sales channels
- Multi-day and single-day event support
- Maximum ticket purchase limit per registration
- Time-based pricing tiers (Early Bird)

### Promo Codes
- Create unique discount codes
- Set usage limits and expiration dates
- Percentage, fixed amount, or free ticket discounts
- Track usage and performance per code

### On-Site Sales
- Walk-in ticket sales with cash/card payments
- Multi-ticket purchases for single customers
- Real-time inventory synchronization
- Receipt generation and printing

### Ticket Counter Management
- Multiple ticket counter support
- Cash session tracking with opening/closing counts
- Automatic discrepancy detection
- Cashier assignment and performance reports

### Online Payments
- Seamless Stripe integration
- Multi-currency support
- Tax configuration options

### QR Code Check-In System
- Auto-generated unique QR codes
- Mobile-friendly QR code scanning for check-ins
- Check-in agent role with mobile access
- Real-time attendee check-in tracking

### AI After-Sales Chatbot
- AI-powered support chatbot for customer assistance
- Natural language conversation with intent classification
- Quick action buttons for common support requests
- Session-based conversation tracking
- Handles:
  - Ticket resending
  - Order inquiries
  - Check-in status
  - Shipment tracking
  - And more
- OTP verification for sensitive operations
- Automatic fallback to contact form for unhandled requests

### Attendee Management
- Complete attendee list dashboard
- Bulk attendee import/export
- Save details for all registrants or primary registrant only
- Send tickets via email and WhatsApp

### Analytics & Reporting
- Event statistics (registrations, check-ins)
- Sales & revenue reporting by channel (online vs on-site)
- Cashier performance tracking
- Data export (CSV/Excel)

### Merchandise Shop
- Add and manage products
- Track product orders
- Shipping options (pickup or delivery)

### Custom Form Builder
- Create custom registration forms
- Add fields: text, dropdowns, checkboxes, etc.

### Role-Based Access Control
- Admin verification panel for organizers
- Organizer dashboard with full event control
- Roles: Admin, Organizer, Cashier, Check-in Agent
- Counter-specific access restrictions
- Homepage banner carousel (fully customizable)

### Admin Customization & Branding
- Customize site appearance and theme colors (light/dark modes)
- Upload and configure header logo with positioning options
- Manage homepage banner carousel with scheduling
- Customize footer layout, company info, and social links
- Configure organizer dashboard banners
- Full white-label branding capabilities

### Other Features
- Visitor tracking (name, email, phone from landing-page form)
- Cart recovery with automated email reminders
- Clean, modern UI with dark/light theme support
- Customizable date format
- Profile settings (update info, theme toggle, account deletion)
